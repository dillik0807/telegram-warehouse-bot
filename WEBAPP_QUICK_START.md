# Быстрый старт веб-приложения

## Запуск

1. Откройте терминал в папке проекта

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите:
```
http://localhost:5000
```

4. Войдите используя логин и пароль из Telegram бота

## Важно!

- При первом входе после обновления нужно выйти и войти заново (старая сессия будет очищена автоматически)
- Используйте те же учетные данные, что и для бота
- Сессия действует 24 часа

## Основные функции

### Главное меню:
- **Приходы** - добавление и просмотр поступлений товаров
- **Расходы** - добавление и просмотр вывода товаров
- **Погашения** - учет оплат от клиентов
- **Партнеры** - учет операций с партнерами
- **Отчеты** - различные отчеты и аналитика

### Отчеты:
- **Долги клиентов** - кто сколько должен
- **Отчет за день** - все операции за выбранный день
- **Расход за день** - детальный расход по складам
- **Фактический остаток** - текущие остатки на складах

### Управление (только админ):
- **Склады** - управление складами
- **Клиенты** - управление клиентами
- **Пользователи** - управление пользователями

## Остановка приложения

Нажмите `Ctrl+C` в терминале где запущено приложение
