# Сводка по системе учета товаров

## ✅ Реализованные функции

### 🔐 Система аутентификации
- Вход через логин и пароль
- Хеширование паролей (SHA-256)
- Управление сессиями
- Кнопка "Выход" в главном меню
- **Логин по умолчанию:** admin
- **Пароль по умолчанию:** admin123

### 👥 Система ролей (4 уровня)
- 👑 **Управляющий** - полный доступ ко всем функциям
- 📊 **Менеджер** - управление товарами, просмотр отчетов
- 🏢 **Завсклад** - управление складом + функции менеджера
- 💰 **Кассир** - только просмотр и продажи

### 🏢 Привязка к группам складов
- Пользователи видят только склады своей группы
- Админ видит все склады
- Гибкое назначение групп при создании пользователя

### 📦 Детальный учет прихода товара (8 полей)
1. 📅 **Дата прихода** - обязательно
2. 🚂 **№ Вагона** - опционально
3. 🏭 **Фирма** - выбор из списка, обязательно
4. 🏢 **Склад** - выбор из доступных, обязательно
5. 📦 **Товар** - выбор или новый, обязательно
6. 📄 **Количество по документу** - обязательно
7. ✅ **Количество по факту** - обязательно
8. 📝 **Примечания** - опционально

### ⚠️ Контроль расхождений
- Автоматическое вычисление разницы между документом и фактом
- Предупреждение при расхождениях
- Остатки обновляются по фактическому количеству

### ⚙️ Управление справочниками
- 🏢 **Склады** с подгруппами
- 📦 **Товары**
- 🏭 **Фирмы**
- 👤 **Клиенты**
- 💰 **Цены** (в долларах)
- 📊 **Коалиции**

### 📊 Отчеты и история
- История всех операций
- Отчеты по товарам
- Статистика по приходам
- Топ товаров

### 📝 Логирование
- Все действия пользователей записываются
- Таблица access_logs для аудита
- Информация о пользователе, оформившем приход

## 📁 Структура базы данных

### Основные таблицы
- `users` - пользователи с ролями и логинами
- `user_sessions` - активные сессии
- `arrivals` - детальная информация о приходах
- `products` - товары и остатки
- `transactions` - история операций
- `warehouses` - склады
- `warehouse_groups` - группы складов
- `firms` - фирмы
- `clients` - клиенты
- `prices` - цены на товары
- `coalitions` - коалиции
- `access_logs` - логи доступа

## 🚀 Запуск системы

### Миграции (по порядку)


