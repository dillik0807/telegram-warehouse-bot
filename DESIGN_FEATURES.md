# 🎨 Особенности современного дизайна

## 🌟 Главные визуальные элементы

### 1. Градиентная навигация
```
Фон: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
Цвет текста: Белый с тенью
Эффект: Плавный переход при наведении
```

### 2. Темная боковая панель
```
Фон: #2d3748 (темно-серый)
Активный элемент: Градиентный фон
Эффект наведения: Сдвиг вправо + градиент
Скроллбар: Кастомный фиолетовый
```

### 3. Карточки действий (Главная страница)
```
Приходы: Фиолетовый градиент (#667eea → #764ba2)
Расходы: Розовый градиент (#f093fb → #f5576c)
Погашения: Голубой градиент (#4facfe → #00f2fe)
Отчеты: Зеленый градиент (#43e97b → #38f9d7)

Эффект: Подъем на 5px при наведении
Тень: Увеличивается при наведении
```

### 4. Современные таблицы
```
Заголовок: Градиентный фон (по разделу)
Строки: Белый фон с границами
Эффект наведения: Градиентная подсветка + масштабирование
Закругление: 15px
Тень: 0 4px 15px rgba(0,0,0,0.1)
```

### 5. Стильные формы
```
Поля ввода: Закругленные (10px)
Граница: 2px solid #e0e0e0
Фокус: Цветная граница + тень
Кнопки: Градиентные с анимацией
```

### 6. Кнопки
```
Основные: Градиентный фон
Удаление: Розовый градиент
Экспорт: Зеленый градиент
Эффект: Масштабирование + тень при наведении
```

## 🎯 Цветовая палитра по разделам

### Приходы (Arrivals)
- **Основной**: #667eea (фиолетовый)
- **Вторичный**: #764ba2 (темно-фиолетовый)
- **Использование**: Заголовки, кнопки, таблицы

### Расходы (Departures)
- **Основной**: #f093fb (светло-розовый)
- **Вторичный**: #f5576c (розово-красный)
- **Использование**: Заголовки, кнопки, таблицы

### Погашения (Payments)
- **Основной**: #4facfe (голубой)
- **Вторичный**: #00f2fe (бирюзовый)
- **Использование**: Заголовки, кнопки, таблицы

### Партнеры (Partners)
- **Основной**: #43e97b (зеленый)
- **Вторичный**: #38f9d7 (бирюзово-зеленый)
- **Использование**: Заголовки, кнопки, таблицы

### Отчеты (Reports)
- **Основной**: #667eea (фиолетовый)
- **Вторичный**: #764ba2 (темно-фиолетовый)
- **Использование**: Карточки отчетов

## 🎬 Анимации

### Карточки
```css
transform: translateY(-5px);
box-shadow: 0 8px 25px rgba(0,0,0,0.15);
transition: all 0.3s ease;
```

### Кнопки
```css
transform: scale(1.05);
box-shadow: 0 4px 15px rgba(102, 126, 234, 0.3);
transition: all 0.3s ease;
```

### Строки таблиц
```css
background: linear-gradient(90deg, rgba(102, 126, 234, 0.05), transparent);
transform: scale(1.01);
transition: all 0.3s ease;
```

### Элементы меню
```css
padding-left: 2rem;
background: linear-gradient(90deg, rgba(102, 126, 234, 0.1), transparent);
transition: all 0.3s ease;
```

## 💡 Советы по использованию

1. **Единообразие**: Используйте одинаковые градиенты для одного раздела
2. **Плавность**: Все анимации должны быть плавными (0.3s ease)
3. **Контраст**: Обеспечьте достаточный контраст для читаемости
4. **Тени**: Используйте умеренно для создания глубины
5. **Закругления**: Сохраняйте единый стиль закруглений
