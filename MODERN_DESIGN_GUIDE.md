# 🎨 Руководство по современному дизайну веб-приложения

## Обзор изменений

Веб-приложение было полностью модернизировано с использованием современных дизайн-паттернов, градиентов, анимаций и улучшенного пользовательского опыта.

## 🌈 Цветовая схема

### Основные цвета
- **Primary Color**: `#667eea` (фиолетовый)
- **Secondary Color**: `#764ba2` (темно-фиолетовый)
- **Градиент навигации**: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`

### Градиенты для разделов
- **Приходы**: `linear-gradient(135deg, #667eea, #764ba2)` - фиолетовый
- **Расходы**: `linear-gradient(135deg, #f093fb, #f5576c)` - розовый
- **Погашения**: `linear-gradient(135deg, #4facfe, #00f2fe)` - голубой
- **Отчеты**: `linear-gradient(135deg, #43e97b, #38f9d7)` - зеленый

## 🎨 CSS классы

### Карточки действий
```css
.action-card {
    border: none;
    border-radius: 15px;
    overflow: hidden;
    transition: all 0.3s ease;
    box-shadow: 0 4px 15px rgba(0,0,0,0.1);
}
.action-card:hover {
    transform: translateY(-5px);
    box-shadow: 0 8px 25px rgba(0,0,0,0.15);
}
```

### Современные таблицы
```css
.modern-table {
    background: white;
    border-radius: 15px;
    overflow: hidden;
    box-shadow: 0 4px 15px rgba(0,0,0,0.1);
}
.modern-table tbody tr:hover {
    background: linear-gradient(90deg, rgba(102, 126, 234, 0.05), transparent);
    transform: scale(1.01);
}
```


## 🚀 Анимации

### Эффекты при наведении
- **Карточки**: Подъем на 5px вверх
- **Кнопки**: Масштабирование 1.05x
- **Строки таблиц**: Масштабирование 1.01x
- **Элементы списка**: Сдвиг вправо

### Переходы
- Все анимации используют `transition: all 0.3s ease`
- Плавные изменения цвета, размера и позиции
