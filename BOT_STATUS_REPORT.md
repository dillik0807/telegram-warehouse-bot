## Как запустить бота

### Вариант 1: Прямой запуск
```bash
python bot.py
```

### Вариант 2: Через nohup (фоновый режим)
```bash
nohup python bot.py > bot.log 2>&1 &
```

### Вариант 3: Через systemd (Linux)
Создать service файл для автозапуска

### Вариант 4: Через Docker
Создать Dockerfile и docker-compose.yml
