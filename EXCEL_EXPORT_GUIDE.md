# Руководство по экспорту в Excel

## Доступные экспорты в веб-приложении

### 1. 💳 Долги клиентов
**Где:** Отчеты → Долги клиентов → Кнопка "Экспорт в Excel"

**Содержимое:**
- Список всех клиентов с долгами
- Сумма продаж по каждому клиенту
- Сумма погашений
- Остаток долга
- Итоговые суммы

**Формат:**
- Заголовки с цветным фоном
- Числовое форматирование для сумм
- Итоговая строка с желтым фоном

### 2. 👤 Карточка клиента
**Где:** Отчеты → Карточка клиента → Выбрать клиента → Кнопка "Экспорт в Excel"

**Содержимое:**
- **Лист 1 "Покупки"**: Все покупки клиента (дата, склад, товар, количество, цена, сумма)
- **Лист 2 "Погашения"**: Все погашения (дата, сомони, курс, сумма $, примечания)
- **Лист 3 "Сводка"**: Итоговые данные (сумма продаж, погашения, остаток долга)

**Формат:**
- 3 листа в одном файле
- Заголовки с цветным фоном
- Числовое форматирование

### 3. 📅 Отчет за день
**Где:** Отчеты → Отчет за день → Выбрать дату → Кнопка "Экспорт в Excel"

**Содержимое:**
- **Лист 1 "Приходы"**: Все приходы за выбранный день
- **Лист 2 "Расходы"**: Все расходы за выбранный день

**Формат:**
- 2 листа в одном файле
- Заголовки с цветным фоном
- Детальная информация по каждой операции

## Как использовать

### Шаг 1: Открыть нужный отчет
Перейдите в раздел "Отчеты" и выберите нужный отчет из списка.

### Шаг 2: Настроить параметры (если нужно)
Для некоторых отчетов можно выбрать:
- Дату (для отчета за день)
- Клиента (для карточки клиента)
- Период (для уведомлений о долгах)

### Шаг 3: Нажать кнопку "Экспорт в Excel"
Кнопка обычно находится в правом верхнем углу или внизу страницы.

### Шаг 4: Сохранить файл
Браузер автоматически скачает файл Excel с именем, содержащим:
- Тип отчета
- Дату/имя клиента (если применимо)
- Дату и время создания файла

## Примеры имен файлов

- `client_debts_20260225_143022.xlsx` - Долги клиентов
- `client_card_ИванИванов_20260225_143045.xlsx` - Карточка клиента
- `daily_report_2026-02-25.xlsx` - Отчет за день

## Работа с файлами Excel

### Открытие файлов
Файлы можно открыть в:
- Microsoft Excel
- LibreOffice Calc
- Google Sheets (загрузить файл)
- Numbers (на Mac)

### Редактирование
После экспорта вы можете:
- Добавлять свои комментарии
- Создавать дополнительные расчеты
- Строить графики и диаграммы
- Фильтровать и сортировать данные
- Печатать отчеты

### Форматирование
Все экспортированные файлы уже имеют:
- ✅ Заголовки с цветным фоном
- ✅ Правильное форматирование чисел
- ✅ Подходящую ширину колонок
- ✅ Итоговые строки (где применимо)

## Советы

1. **Регулярный экспорт**: Экспортируйте важные отчеты регулярно для архива
2. **Резервные копии**: Сохраняйте файлы в надежном месте
3. **Анализ данных**: Используйте Excel для дополнительного анализа
4. **Печать**: Файлы готовы для печати без дополнительной настройки
5. **Отправка**: Легко отправить отчет по email или мессенджеру

## Ограничения

- Максимум 10,000 записей в одном экспорте
- Файлы создаются в формате .xlsx (Excel 2007+)
- Требуется современный браузер с поддержкой скачивания файлов

## Устранение проблем

**Файл не скачивается:**
- Проверьте настройки браузера (разрешены ли скачивания)
- Проверьте место на диске
- Попробуйте другой браузер

**Файл не открывается:**
- Убедитесь, что установлена программа для работы с Excel
- Проверьте, что файл скачался полностью
- Попробуйте открыть в Google Sheets онлайн

**Неправильные данные:**
- Обновите страницу и попробуйте снова
- Проверьте выбранные параметры (дата, клиент и т.д.)
- Убедитесь, что у вас есть права доступа к данным
