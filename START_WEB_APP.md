# Быстрый старт веб-приложения

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements_web.txt
```

## Шаг 2: Запуск приложения

```bash
python app.py
```

## Шаг 3: Открыть в браузере

Перейдите по адресу: **http://localhost:5000**

## Вход в систему

Используйте логин и пароль пользователя из базы данных.

### Создание первого пользователя (если нужно)

Через Telegram бота или напрямую в базе данных:

```sql
-- Пример создания администратора
-- Пароль: admin123 (хеш SHA256)
INSERT INTO users (user_id, login, password, username, role, is_active)
VALUES (1, 'admin', '240be518fabd2724ddb6f04eeb1da5967448d7e831c08c8fa822809f74c720a9', 'Администратор', 'admin', true);
```

## Доступные страницы

- `/` - Главная страница
- `/arrivals` - Список приходов
- `/arrivals/add` - Добавить приход
- `/departures` - Список расходов
- `/departures/add` - Добавить расход
- `/payments` - Список погашений
- `/payments/add` - Добавить погашение
- `/partners` - Список партнеров
- `/reports` - Отчеты
- `/reports/client_debts` - Долги клиентов
- `/management` - Управление (только для админов)

## Примечания

- Приложение использует ту же базу данных, что и Telegram бот
- Все изменения синхронизируются между ботом и веб-приложением
- Для продакшена измените `app.secret_key` в app.py
- Рекомендуется использовать HTTPS в продакшене
