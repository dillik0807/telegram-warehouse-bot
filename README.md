# Telegram бот для учета товаров

Бот для учета прихода и вывода товаров на складе с системой ролей и аутентификацией.

## Функции

### Основные
- ➕ Приход товара
- ➖ Вывод товара
- 📦 Просмотр остатков
- 📊 История операций
- 📋 Отчеты

### Управление (для администраторов)
- 👥 Управление пользователями с ролями
- 🏢 Управление складами и подгруппами
- 📦 Управление товарами
- 🏭 Управление фирмами
- 👤 Управление клиентами
- 💰 Управление ценами
- 📊 Управление коалициями

### Система ролей
- 👑 Управляющий - полный доступ
- 📊 Менеджер - управление товарами
- 🏢 Завсклад - управление складом
- 💰 Кассир - просмотр и продажи

### Безопасность
- 🔐 Вход через логин/пароль
- 🔒 Хеширование паролей (SHA-256)
- 🚪 Выход из системы
- 📝 Логирование всех действий

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте PostgreSQL базу данных на Railway.com:
   - Зайдите на https://railway.com/
   - Создайте новый проект
   - Добавьте PostgreSQL сервис
   - Скопируйте DATABASE_URL из переменных окружения

3. Создайте файл `.env` и добавьте данные:
```
BOT_TOKEN=your_bot_token_here
DATABASE_URL=postgresql://user:password@host:port/database
```

4. Получите токен бота у @BotFather в Telegram

5. Запустите бота:
```bash
python bot.py
```

## Деплой на Railway.com

1. Создайте новый проект на Railway
2. Подключите GitHub репозиторий или загрузите код
3. Добавьте PostgreSQL сервис
4. Добавьте переменные окружения:
   - `BOT_TOKEN` - токен вашего бота
   - `DATABASE_URL` - автоматически создается Railway
5. Railway автоматически запустит бота

## Использование

### Первый вход
1. Запустите бота командой `/start`
2. Введите логин: `admin`
3. Введите пароль: `admin123`
4. Вы войдете в систему как Управляющий

⚠️ **Важно**: Рекомендуется сменить пароль после первого входа!

### Выход из системы
В главном меню нажмите **🚪 Выход** для выхода из системы.

### Документация
- `LOGIN_GUIDE.md` - инструкция по входу и использованию
- `ROLES_GUIDE.md` - подробное описание ролей и прав доступа

## Миграции базы данных

После установки запустите миграции:
```bash
python migrate_db.py          # Создание основных таблиц
python migrate_roles.py       # Добавление системы ролей
python migrate_login_system.py # Добавление логин/пароль
```
3. Следуйте инструкциям бота
