# Инструкция по входу в систему

## Первый вход

1. Запустите бота командой `/start`
2. Введите логин: `admin`
3. Введите пароль: `admin123`
4. Вы войдете в систему как Управляющий

## Главное меню

После входа вам доступны следующие функции:

### 📦 Управление товарами
- **➕ Приход товара** - добавление товара на склад
- **➖ Вывод товара** - списание товара со склада
- **📦 Остатки** - просмотр текущих остатков
- **📊 История** - история всех операций

### 👥 Управление (только для админов)
- **👥 Пользователи** - управление пользователями и ролями
- **⚙️ Управление** - управление справочниками (склады, товары, фирмы, клиенты, цены, коалиции)
- **📋 Отчет** - статистика по товарам

### 🚪 Выход
- **🚪 Выход** - выход из системы (удаление сессии)

## Добавление нового пользователя

Только администраторы могут добавлять пользователей:

1. Выберите **👥 Пользователи**
2. Нажмите **➕ Добавить пользователя**
3. Введите логин (минимум 3 символа)
4. Введите пароль (минимум 4 символа)
5. Введите имя пользователя (ФИО)
6. Выберите роль:
   - 👑 Управляющий - полный доступ
   - 📊 Менеджер - управление товарами
   - 🏢 Завсклад - управление складом
   - 💰 Кассир - только просмотр и продажи
7. Выберите группу складов (если не Управляющий)

## Выход из системы

Для выхода из системы:
1. В главном меню нажмите **🚪 Выход**
2. Ваша сессия будет удалена
3. Для повторного входа используйте `/start`

## Безопасность

- Пароли хранятся в зашифрованном виде (SHA-256)
- Сессии автоматически обновляются при активности
- Рекомендуется сменить пароль по умолчанию после первого входа
- Не передавайте свои логин и пароль другим пользователям

## Смена пароля

Функция смены пароля будет добавлена в следующей версии.
Пока для смены пароля обратитесь к администратору.
