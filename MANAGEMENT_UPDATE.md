# ✅ Обновление раздела "Управление"

## Проблема
Раздел "Управление" в веб-приложении не соответствовал телеграм боту. В боте было 8 разделов, а в веб-приложении только 3.

## Решение
Добавлены все недостающие разделы управления для полного соответствия с телеграм ботом.

## 📋 Разделы управления

### В телеграм боте:
1. 🏢 Склады
2. 📦 Товары
3. 🏭 Фирмы
4. 👤 Клиенты
5. 💰 Цены
6. 📊 Коалица
7. ✏️ Изменить записи
8. 🗑 Удалить записи

### В веб-приложении (после обновления):
1. ✅ 🏢 Склады - Управление складами и подгруппами
2. ✅ 📦 Товары - Список всех товаров в системе
3. ✅ 🏭 Фирмы - Добавление и удаление фирм
4. ✅ 👤 Клиенты - Управление клиентами
5. ✅ 💰 Цены - Установка и удаление цен
6. ✅ 📊 Коалица - Добавление и удаление коалиц
7. ✅ 👥 Пользователи - Управление пользователями системы

**Примечание**: Функции "Изменить записи" и "Удалить записи" реализованы через кнопки удаления в таблицах (только для админов).

## 🆕 Добавленные файлы

### Шаблоны HTML:
1. **templates/manage_products.html** - Управление товарами
2. **templates/manage_firms.html** - Управление фирмами
3. **templates/manage_prices.html** - Управление ценами
4. **templates/manage_coalitions.html** - Управление коалицей

### Обновленные файлы:
1. **templates/management.html** - Главная страница управления (добавлены новые карточки)
2. **app.py** - Добавлены маршруты для новых разделов

## 🎯 Функциональность

### 📦 Управление товарами
- Просмотр списка всех товаров
- Товары добавляются автоматически при создании приходов
- Информационное сообщение о способе добавления

### 🏭 Управление фирмами
- ➕ Добавление новой фирмы (модальное окно)
- 📋 Список всех фирм
- 🗑 Удаление фирмы (с проверкой использования)
- Защита от удаления используемых фирм

### 💰 Управление ценами
- 💰 Установка цены для товара (модальное окно)
- 📋 Список всех установленных цен
- Отображение даты установки и пользователя
- 🗑 Удаление цены

### 📊 Управление коалицей
- ➕ Добавление новой коалицы (модальное окно)
- 📋 Список всех коалиц
- 🗑 Удаление коалицы (с проверкой использования)
- Защита от удаления используемых коалиц

## 📊 Маршруты в app.py

### Товары:
- `GET /management/products` - Список товаров
- `POST /management/products/add` - Добавить товар
- `POST /management/products/delete` - Удалить товар

### Фирмы:
- `GET /management/firms` - Список фирм
- `POST /management/firms/add` - Добавить фирму
- `GET /management/firms/delete/<id>` - Удалить фирму

### Цены:
- `GET /management/prices` - Список цен
- `POST /management/prices/add` - Установить цену
- `GET /management/prices/delete/<id>` - Удалить цену

### Коалица:
- `GET /management/coalitions` - Список коалиц
- `POST /management/coalitions/add` - Добавить коалицу
- `GET /management/coalitions/delete/<id>` - Удалить коалицу

## 🚀 Использование

1. Войдите в систему как администратор
2. Перейдите в раздел "⚙️ Управление" в боковом меню
3. Выберите нужный раздел управления
4. Используйте кнопки для добавления/удаления записей

## 📝 Примечания

- Товары добавляются автоматически при создании приходов
- Удаление фирм и коалиц возможно только если они не используются
- Все действия логируются в системе
- Модальные окна для удобного добавления записей
