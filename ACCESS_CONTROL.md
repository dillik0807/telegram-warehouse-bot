# Контроль доступа по ролям

## Обзор

Система контроля доступа обеспечивает разделение данных между пользователями в зависимости от их роли и группы складов.

## Доступ к данным при оформлении прихода

### 🏭 Фирмы
**Все пользователи видят ВСЕ фирмы**
- 👑 Управляющий - все фирмы
- 📊 Менеджер - все фирмы
- 🏢 Завсклад - все фирмы
- 💰 Кассир - все фирмы

**Причина:** Фирмы-поставщики общие для всей организации

### 🏢 Склады
**Пользователи видят только склады своей группы**
- 👑 Управляющий - ВСЕ склады
- 📊 Менеджер - только склады своей группы
- 🏢 Завсклад - только склады своей группы
- 💰 Кассир - только склады своей группы

**Причина:** Каждый завсклад отвечает только за свои склады

### 📦 Товары
**При оформлении прихода - все видят ВСЕ товары из справочника**
- 👑 Управляющий - все товары
- 📊 Менеджер - все товары
- 🏢 Завсклад - все товары
- 💰 Кассир - все товары

**При просмотре остатков - видят только товары своих складов**
- 👑 Управляющий - товары со ВСЕХ складов
- 📊 Менеджер - товары только со складов своей группы
- 🏢 Завсклад - товары только со складов своей группы
- 💰 Кассир - товары только со складов своей группы

**Причина:** При приходе можно оприходовать любой товар на свой склад, даже если его там раньше не было

## Примеры доступа

### Пример 1: Завсклад группы "Север"

**Назначение:**
- Роль: Завсклад
- Группа: Север

**Что видит при оформлении прихода:**

1. **Фирмы:** ✅ Все фирмы
   - ООО "Поставщик А"
   - ООО "Поставщик Б"
   - ООО "Поставщик В"

2. **Склады:** ⚠️ Только своей группы
   - Склад Север-1
   - Склад Север-2

3. **Товары:** ✅ Все товары из справочника
   - Цемент
   - Песок
   - Кирпич
   - (может выбрать любой товар для прихода)

**Что видит в остатках:**
- Цемент: 150 шт. (только на складах Север-1 и Север-2)
- Песок: 200 шт.

**Не видит:**
- ❌ Склады группы "Юг"
- ❌ Товары со складов группы "Юг"

### Пример 2: Завсклад группы "Юг"

**Назначение:**
- Роль: Завсклад
- Группа: Юг

**Что видит:**

1. **Фирмы:** ✅ Все фирмы (те же что и у Севера)
2. **Склады:** ⚠️ Только своей группы
   - Склад Юг-1
   - Склад Юг-2
3. **Товары при приходе:** ✅ Все товары из справочника
   - Цемент
   - Песок
   - Кирпич
   - (может выбрать любой)

**Что видит в остатках:**
- Цемент: 80 шт. (только на складах Юг-1 и Юг-2)
- Кирпич: 300 шт.

**Не видит:**
- ❌ Склады группы "Север"
- ❌ Товары со складов группы "Север"

### Пример 3: Управляющий (Админ)

**Назначение:**
- Роль: Управляющий
- Группа: Не назначена (не требуется)

**Что видит:**

1. **Фирмы:** ✅ Все фирмы
2. **Склады:** ✅ ВСЕ склады всех групп
   - Склад Север-1
   - Склад Север-2
   - Склад Юг-1
   - Склад Юг-2
3. **Товары при приходе:** ✅ Все товары из справочника
4. **Товары в остатках:** ✅ Товары со ВСЕХ складов
   - Цемент: 230 шт. (150 + 80)
   - Песок: 200 шт.
   - Кирпич: 300 шт.

## Просмотр остатков

### Кнопка "📦 Остатки"

**Показывает остатки с учетом группы:**

- **Управляющий:** Видит общие остатки со всех складов
- **Завсклад Север:** Видит только остатки на складах группы "Север"
- **Завсклад Юг:** Видит только остатки на складах группы "Юг"

### Расчет остатков

Остатки рассчитываются на основе:
- Всех приходов на склады группы пользователя
- Фактического количества (не по документу)
- Суммируются по названию товара

## Безопасность данных

### Изоляция данных
- Завсклады не видят данные других групп
- Каждая группа работает независимо
- Админ имеет полный обзор

### Логирование
- Все действия записываются с указанием пользователя
- Можно отследить кто и когда оформил приход
- Аудит доступа к данным

## Настройка доступа

### Для назначения группы пользователю:

1. Войдите как Управляющий
2. Выберите **👥 Пользователи**
3. Выберите **🏢 Назначить склад**
4. Выберите пользователя
5. Выберите группу складов

### Важно:
- Пользователь без группы не видит складов и товаров
- Админу группа не нужна (видит всё)
- Группу можно изменить в любой момент

## Рекомендации

1. **Создайте группы по регионам или направлениям:**
   - Группа "Север"
   - Группа "Юг"
   - Группа "Центр"

2. **Назначьте склады в группы:**
   - Все склады региона в одну группу
   - Можно создать подгруппы внутри региона

3. **Назначьте завскладов в группы:**
   - Каждый завсклад отвечает за свою группу
   - Один завсклад может быть только в одной группе

4. **Фирмы остаются общими:**
   - Не нужно дублировать фирмы для каждой группы
   - Все работают с одним списком поставщиков
