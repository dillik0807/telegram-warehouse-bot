# ✅ Добавлены формы управления данными

## Проблема
В разделах управления клиентами, складами и товарами не было возможности добавлять новые записи через веб-интерфейс.

## Решение
Добавлены модальные окна для добавления данных и кнопки удаления во всех разделах управления.

## 📋 Обновленные разделы

### 1. 👤 Управление клиентами
**Файл**: `templates/manage_clients.html`

**Добавлено**:
- ➕ Кнопка "Добавить клиента"
- Модальное окно с формой:
  - Имя клиента (обязательно)
  - Телефон (необязательно)
- 🗑 Кнопка удаления для каждого клиента
- Проверка использования перед удалением
- Современный дизайн с градиентами

**Маршруты в app.py**:
- `POST /management/clients/add` - Добавить клиента
- `GET /management/clients/delete/<id>` - Удалить клиента

### 2. 🏢 Управление складами
**Файл**: `templates/manage_warehouses.html`

**Добавлено**:
- ➕ Кнопка "Добавить склад"
- Модальное окно с формой:
  - Название склада (обязательно)
  - Адрес (необязательно)
  - Подгруппа (выбор из списка, необязательно)
- ➕ Кнопка "Добавить подгруппу"
- Модальное окно для подгруппы:
  - Название подгруппы (обязательно)
- 🗑 Кнопки удаления для складов и подгрупп
- Проверка использования перед удалением
- Разделение на две таблицы: Склады и Подгруппы

**Маршруты в app.py**:
- `POST /management/warehouses/add` - Добавить склад
- `GET /management/warehouses/delete/<id>` - Удалить склад
- `POST /management/warehouse_groups/add` - Добавить подгруппу
- `GET /management/warehouse_groups/delete/<id>` - Удалить подгруппу

## 📊 Проверки перед удалением

### Клиенты
Проверяется использование в таблице `departures`:
```sql
SELECT COUNT(*) FROM departures WHERE client_id = %s
```

### Склады
Проверяется использование в таблице `arrivals`:
```sql
SELECT COUNT(*) FROM arrivals WHERE warehouse_id = %s
```

### Подгруппы складов
Проверяется использование в таблице `warehouses`:
```sql
SELECT COUNT(*) FROM warehouses WHERE group_id = %s
```

### Фирмы
Проверяется использование в таблице `arrivals`:
```sql
SELECT COUNT(*) FROM arrivals WHERE firm_id = %s
```

### Коалиция
Проверяется использование в таблице `departures`:
```sql
SELECT COUNT(*) FROM departures WHERE coalition_id = %s
```

## 🎯 Функциональность

### Добавление данных
1. Нажать кнопку "Добавить"
2. Заполнить форму в модальном окне
3. Нажать "Добавить" в модальном окне
4. Данные сохраняются в БД
5. Отображается сообщение об успехе/ошибке
6. Страница перезагружается с обновленными данными

### Удаление данных
1. Нажать кнопку удаления (🗑)
2. Подтвердить действие в диалоге
3. Проверяется использование записи
4. Если используется - показывается ошибка
5. Если не используется - запись удаляется
6. Отображается сообщение об успехе/ошибке

## 📝 Структура таблиц

### Клиенты (clients)
```sql
- id (PRIMARY KEY)
- name (TEXT)
- phone (TEXT)
```

### Склады (warehouses)
```sql
- id (PRIMARY KEY)
- name (TEXT)
- address (TEXT)
- group_id (FOREIGN KEY → warehouse_groups.id)
```

### Подгруппы складов (warehouse_groups)
```sql
- id (PRIMARY KEY)
- name (TEXT)
```

### Фирмы (firms)
```sql
- id (PRIMARY KEY)
- firm_name (TEXT)
```

### Коалиция (coalitions)
```sql
- id (PRIMARY KEY)
- name (TEXT)
- contact (TEXT)
- created_at (TIMESTAMP)
```

### Цены (prices)
```sql
- id (PRIMARY KEY)
- product_name (TEXT UNIQUE)
- price (DECIMAL)
- updated_at (TIMESTAMP)
```
