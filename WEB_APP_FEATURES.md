# Полный список функций веб-приложения

## ✅ Все функции из Telegram бота реализованы!

### 📥 Приходы
- ✅ Добавление прихода товара
- ✅ Просмотр списка приходов
- ✅ Отчет по приходам (группировка по месяцам)

### 📤 Расходы
- ✅ Добавление расхода товара
- ✅ Просмотр списка расходов
- ✅ Отчет по расходам (группировка по месяцам)
- ✅ Формула: (quantity / 20) * price

### 💰 Погашения
- ✅ Добавление погашения
- ✅ Просмотр списка погашений
- ✅ Отчет по погашениям
- ✅ Формула: somoni / курс

### 🤝 Партнеры
- ✅ Просмотр списка партнеров
- ✅ Отчет по партнерам

### 📊 Отчеты

#### Основные отчеты:
1. **💳 Долги клиентов**
   - Сумма продаж по каждому клиенту
   - Сумма погашений
   - Остаток долга
   - Итоговые суммы

2. **📅 Отчет за день**
   - Приходы за выбранный день
   - Расходы за выбранный день
   - Выбор даты (сегодня, вчера, позавчера)

3. **📤 Расход за день**
   - Группировка по складам
   - Группировка по товарам
   - Вес в тоннах
   - Общий итог

4. **🏭 Фактический остаток**
   - Группировка по коалициям
   - Группировка по складам
   - Группировка по товарам
   - Вес в тоннах
   - Итоги на всех уровнях

#### Детальные отчеты:
5. **➕ Отчет по приходам**
   - Группировка по месяцам
   - Количество записей
   - Общий вес в тоннах
   - Итоги за год

6. **➖ Отчет по расходам**
   - Группировка по месяцам
   - Количество записей
   - Общая сумма в долларах
   - Итоги за год

7. **👤 Карточка клиента**
   - Выбор клиента из списка
   - Все покупки клиента
   - Все погашения клиента
   - Сводка: продажи, погашения, остаток

8. **📢 Уведомления о долгах**
   - Выбор периода (сегодня, вчера, 7/14/30 дней назад)
   - Покупки на выбранную дату
   - Текущие долги всех клиентов

9. **🚂 Итоги вагонов**
   - Группировка по складам
   - Группировка по товарам
   - Количество вагонов
   - По документам vs Фактически
   - Разница
   - Вес в тоннах
   - Общий итог

10. **📥 Список приходов**
    - Детальный список всех приходов
    - Фильтрация по складам (для пользователей)
    - Все склады (для админов)

11. **📤 Список расходов**
    - Детальный список всех расходов
    - Фильтрация по складам (для пользователей)
    - Все склады (для админов)

12. **💰 Список погашений**
    - Детальный список всех погашений
    - Клиент, сумма, курс, дата

13. **🤝 Список партнеров**
    - Детальный список операций с партнерами
    - Клиент, сумма, курс, дата

### ⚙️ Управление (только для администраторов)

1. **🏢 Управление складами**
   - Просмотр списка складов
   - Добавление складов
   - Редактирование складов
   - Удаление складов

2. **👤 Управление клиентами**
   - Просмотр списка клиентов
   - Добавление клиентов
   - Редактирование клиентов
   - Удаление клиентов

3. **👥 Управление пользователями**
   - Просмотр списка пользователей
   - Добавление пользователей
   - Изменение ролей
   - Назначение складов
   - Активация/деактивация

### 🔐 Безопасность

- ✅ Система входа с логином и паролем
- ✅ Хеширование паролей (SHA-256)
- ✅ Сессии с автоматическим истечением (24 часа)
- ✅ Проверка прав доступа на каждой странице
- ✅ Разделение прав: admin, manager, user
- ✅ Фильтрация данных по складам для пользователей

### 🎨 Интерфейс

- ✅ Современный дизайн на Bootstrap 5
- ✅ Адаптивная верстка (работает на всех устройствах)
- ✅ Удобная навигация
- ✅ Карточки для быстрого доступа
- ✅ Таблицы с сортировкой
- ✅ Формы с валидацией
- ✅ Уведомления об успехе/ошибках

### 📱 Совместимость

- ✅ Работает на компьютерах
- ✅ Работает на планшетах
- ✅ Работает на смартфонах
- ✅ Все современные браузеры

## 🎯 Полное соответствие Telegram боту

Все функции из Telegram бота теперь доступны в веб-приложении:
- ✅ Все отчеты
- ✅ Все формулы расчетов
- ✅ Все права доступа
- ✅ Вся функциональность управления

## 🚀 Преимущества веб-приложения

1. **Удобный доступ** - работает из любого браузера
2. **Большой экран** - удобнее просматривать таблицы и отчеты
3. **Быстрая навигация** - все функции в одном месте
4. **Печать отчетов** - можно распечатать любой отчет
5. **Экспорт данных** - легко копировать данные из таблиц
6. **Многозадачность** - можно открыть несколько вкладок

## 📝 Примечания

- Используется та же база данных, что и Telegram бот
- Все изменения синхронизируются между ботом и веб-приложением
- Можно работать одновременно в боте и веб-приложении
