# Руководство по оформлению прихода товара

## Обзор

Система детального учета прихода товара позволяет фиксировать все важные данные о поступлении товара на склад.

## Поля прихода товара

При оформлении прихода товара необходимо заполнить следующие поля:

### 1. 📅 Дата прихода
- Формат: ДД.ММ.ГГГГ (например: 23.02.2026)
- Можно выбрать "Сегодня" для текущей даты
- Обязательное поле

### 2. 🚂 № Вагона
- Номер вагона, в котором прибыл товар
- Необязательное поле (можно пропустить)
- Полезно для железнодорожных поставок

### 3. 🏭 Фирма
- Выбор из списка зарегистрированных фирм
- Фирма-поставщик товара
- Обязательное поле
- Если нужной фирмы нет, добавьте её в меню "Управление → Фирмы"

### 4. 🏢 Склад
- Выбор из списка доступных складов
- Склад, на который поступает товар
- Обязательное поле
- Пользователи видят только склады своей группы (кроме админов)

### 5. 📦 Товар
- Выбор из списка существующих товаров
- Или добавление нового товара
- Обязательное поле
- Название товара, который поступает

### 6. � Количество по документу
- Количество товара согласно накладной/документам
- Обязательное поле
- Целое число больше 0

### 7. ✅ Количество по факту
- Фактически принятое количество товара
- Обязательное поле
- Целое число больше 0
- Может отличаться от количества по документу

### 8. 📝 Примечания
- Дополнительная информация о приходе
- Необязательное поле (можно пропустить)
- Например: "Повреждена упаковка", "Срочная поставка"

## Процесс оформления прихода

1. В главном меню выберите **➕ Приход товара**
2. Последовательно заполните все поля
3. После заполнения всех полей приход будет автоматически сохранен
4. Вы получите подтверждение с ID прихода и информацией о расхождениях (если есть)

## Автоматические действия

При оформлении прихода система автоматически:
- Обновляет остатки товара на складе (по фактическому количеству)
- Добавляет запись в историю транзакций
- Сохраняет детальную информацию о приходе
- Вычисляет расхождение между документом и фактом
- Фиксирует пользователя, оформившего приход

## Расхождения

Если количество по факту отличается от количества по документу:
- Система автоматически вычисляет разницу
- Показывает предупреждение о расхождении
- Сохраняет обе цифры для анализа
- Остатки обновляются по фактическому количеству

## Пример

```
📅 Дата: 23.02.2026
🚂 № Вагона: 12345
🏭 Фирма: ООО "Поставщик"
🏢 Склад: Центральный склад
📦 Товар: Цемент М500
 По документу: 1000 шт.
✅ По факту: 995 шт.
📝 Примечания: 5 мешков повреждены

Результат:
✅ Приход оформлен!
⚠️ Расхождение: -5 шт.
📋 ID прихода: 123
```

## Права доступа

Оформлять приход товара могут:
- 👑 Управляющий
- 📊 Менеджер
- 🏢 Завсклад

Кассиры не имеют доступа к оформлению прихода.

## Просмотр истории приходов

История всех приходов доступна в меню **📊 История**.
Для детального просмотра конкретного прихода используйте ID прихода.

## Отчеты

Статистика по приходам включает:
- Общее количество приходов
- Суммарное количество товара
- Общее расхождение между документами и фактом
- Фильтрация по датам, складам, фирмам
